# Lusitania Markets

A Snowflake-native Streamlit application for monitoring Portuguese financial markets using Trading Economics data.

## Features
- Momentum snapshot (daily / weekly / monthly)
- Intraday risk visualization
- Cross-sectional market comparison
- Built directly inside Snowflake using Snowpark + Streamlit

## Data Source
- Trading Economics (Portugal-only trial)
- Snowflake Marketplace

## Tech Stack
- Snowflake
- Snowpark
- Streamlit
- Altair
